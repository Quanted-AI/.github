## About Quanted

Quanted centralises external vendor data into research intelligence front-office teams can query at any point in their journey from idea to investment committee.

Our developer tools and experiments live across these repos. The platform itself sits at [query.quanted.com](https://query.quanted.com).

## What you'll find here

A mix of developer tooling and references, including:

- **[`QAgents`](https://github.com/Quanted-AI/QAgents)**. Type-safe Python SDK for building agentic workflows with Pydantic I/O, composable primitives (Pipeline, Router, Loop, Parallel), and hierarchical orchestration.
- Platform-adjacent tooling and connectors.
- Examples, demos, and other open releases.

More gets added as the platform expands.
